# ProjetBordeciel
Un projet VR d'Astronomie amateur dans le cadre du cours VR Immersive de l'ESME

Ci-joint se trouvent :

BookController.cs, un script pour manipuler le livre interactif.
ConstellationData.cs, un script pour créer les Scriptable Objects des constellations (code, nom d'affichage, image et description).
ConstellationRenderer.cs, un script pour créer les constellations en elles-même en fonction de constellations_2by2.csv.
Star.cs, un script pour instancier les données des étoiles (nom, hip, code de constellation).
StarGenerator.cs, un script pour générer les étoiles en fonction de leur RA, DEC et Magnitude en fonction de hyg_v42.csv.
